# City of Cape Town Data Analytics Hub Challenge

## Technical Assessment for Data Product Coordinator 

Welcome to the technical assessment for the Data Product Coordinator position at the City's Data Analytics Hub. We appreciate you taking the time to complete this challenge.

The goal is to assess your coordination, planning, and communication skills as they relate to supporting a technical data team, based on the provided scenarios.

### Part 3: Practical Skill - Technical Communication (10 Minutes)

**Context:** The data engineering team has just finished a major update to a key internal dashboard. This update will change the way users access data. The team has provided you with the following "technical note" and asked you to help distribute it as part of the change management process.

**Technical Note:**
> "Tonight at 23:00, we are sunsetting the old dashboard auth module and migrating to the new SSO-based OIDC protocol. Users’ cached credentials will be invalidated. They must clear their browser cache (Ctrl+F5) and re-authenticate via the new IdP. Expect potential token expiry errors post-migration. See wiki-page DATA-291 for full spec.”
